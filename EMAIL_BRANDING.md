# Email branding & deliverability — Agendalix

Two goals:
1. **Show the Agendalix logo** as the sender's avatar in Gmail / Apple Mail / Outlook.
2. **Stay out of spam** — pass SPF, DKIM, DMARC.

---

## 1 · Logo as sender avatar

Email clients pick the sender's profile image from one of three sources, in this order:

| Source | Coverage | Cost | Setup |
|---|---|---|---|
| **Gravatar** | Gmail web, ProtonMail, Slack, GitHub, many others | Free | Sign up at gravatar.com with `contacto@agendalix.com` and upload `public/favicon.svg` (or a 512×512 PNG export). |
| **BIMI** | Gmail mobile/web (with VMC), Apple Mail, Yahoo, Fastmail | Free DNS, paid VMC (~$1k/yr from DigiCert/Entrust) | See below. |
| **Inline header** | Body of every message | Already done | The branded HTML in `src/lib/mail.ts` embeds `cid:agendalix-logo` at the top of every outbound mail. |

### Gravatar (do this first — 3 minutes)
1. Go to https://gravatar.com → sign up with `contacto@agendalix.com`.
2. Confirm the email.
3. Upload the Agendalix mark (export `public/favicon.svg` to PNG at 512×512).
4. Done. Gmail will start showing it within ~24 h.

### BIMI (do later — full brand display in Gmail)
A BIMI-compliant SVG is already deployed at `https://agendalix.com/bimi.svg`. To activate:

```dns
default._bimi.agendalix.com.  IN  TXT  "v=BIMI1; l=https://agendalix.com/bimi.svg; a=https://agendalix.com/vmc.pem"
```

> The `a=` parameter is optional but **required for Gmail to display the logo**. It points to a Verified Mark Certificate (VMC) — buy from DigiCert/Entrust if you want full Gmail support. Without it, Yahoo/Apple still display.

BIMI also requires DMARC at `p=quarantine` or `p=reject` (see step 2.3).

## 2 · Deliverability — SPF, DKIM, DMARC

> Replace `1.2.3.4` with the public IP of the SMTP server (`MAIL_SMTP_HOST`).
> If you use a 3rd-party SMTP (SendGrid, Mailgun, Resend, Postmark…), use their published SPF/DKIM values instead.

### 2.1 SPF — authorize the sending IP

```dns
agendalix.com.  IN  TXT  "v=spf1 mx ip4:1.2.3.4 -all"
```

`-all` (hard fail) is safer once you're sure all sending sources are listed.
Use `~all` (soft fail) while testing.

### 2.2 DKIM — sign every outbound message

DKIM is generated by your mail server, not by Next.js. Steps depend on the SMTP backend:

- **Postfix + OpenDKIM (self-hosted)**:
  ```bash
  sudo apt install opendkim opendkim-tools
  sudo mkdir -p /etc/opendkim/keys/agendalix.com
  sudo opendkim-genkey -b 2048 -d agendalix.com -D /etc/opendkim/keys/agendalix.com -s mail -v
  sudo cat /etc/opendkim/keys/agendalix.com/mail.txt   # publish this as a DNS TXT
  ```
  DNS:
  ```dns
  mail._domainkey.agendalix.com.  IN  TXT  "v=DKIM1; k=rsa; p=MIIBIjANBgkqhk..."
  ```

- **3rd-party (SendGrid / Mailgun / Resend)**: copy the CNAME or TXT records from their dashboard.

### 2.3 DMARC — tell receivers what to do on failure

Start in monitoring mode:
```dns
_dmarc.agendalix.com.  IN  TXT  "v=DMARC1; p=none; rua=mailto:dmarc@agendalix.com; pct=100; aspf=r; adkim=r"
```

After 1–2 weeks of clean reports, tighten:
```dns
_dmarc.agendalix.com.  IN  TXT  "v=DMARC1; p=quarantine; rua=mailto:dmarc@agendalix.com; pct=100; sp=quarantine; aspf=s; adkim=s"
```

BIMI requires at least `p=quarantine` with `pct=100`.

## 3 · Hygiene already covered in code

`src/lib/mail.ts` already does:

- ✅ `From: "Agendalix" <contacto@agendalix.com>` — friendly name + address match
- ✅ `Reply-To` header set
- ✅ Stable `Message-ID` with the agendalix.com domain
- ✅ Both `text` and `html` parts (multipart/alternative)
- ✅ `List-Unsubscribe` + `List-Unsubscribe-Post` headers (Gmail/Apple require for bulk)
- ✅ `X-Mailer` and `X-Entity-Ref-ID` for thread tracking
- ✅ Branded HTML wrapper with inline logo (`cid:agendalix-logo`)
- ✅ TLS for SMTP submission (port 465 / `secure: true`)

## 4 · Sanity checks

After DNS propagates (~1 h):

```bash
# Send a test mail to mail-tester.com — get a 10/10 score
echo "test" | mail -s "Agendalix deliverability test" test-XXXXX@mail-tester.com

# Verify SPF/DKIM/DMARC alignment
dig TXT agendalix.com +short
dig TXT _dmarc.agendalix.com +short
dig TXT mail._domainkey.agendalix.com +short

# Verify BIMI
dig TXT default._bimi.agendalix.com +short
```

Or use the web tools:
- https://www.mail-tester.com — full deliverability score
- https://mxtoolbox.com/SuperTool.aspx — DNS lookups
- https://bimigroup.org/bimi-generator/ — BIMI record builder
